## 文件命名规范

每个语言的书籍存放在对应目录，按 `书名-slug/chapter_XXX.md` 格式：

```
content/
├── en/
│   ├── journey-to-the-west/
│   │   ├── chapter_001.md
│   │   └── chapter_002.md
│   └── tao-te-ching/
│       └── chapter_001.md
└── es/
    └── tao-te-ching/
        └── chapter_001.md
```

## 格式要求

每个章节文件使用Markdown格式：
```markdown
# 书名 第X回

正文内容...

---

**原文对照：**
对应中文原文（可选）
```
